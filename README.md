# kernelua x86
porting [kernelua](https://github.com/Penguin-Spy/kernelua) to x86 as well (i.e. normal computers)

# compiling
install: `sudo apt install gnu-efi gcc-mingw-w64-x86-64-posix`  
compile: `make`
## running
install: `sudo apt install qemu-system-x86 ovmf`  
run: `make qemu`
