# kernelua x86
porting [kernelua](https://github.com/Penguin-Spy/kernelua) to x86 as well (i.e. normal computers)

# compiling
build using [a gcc cross compiler targeting `i686-elf`](https://wiki.osdev.org/GCC_Cross-Compiler)
